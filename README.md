# SecureIntel — DNS Guardian 🛡️

An open-source network anomaly detection system that watches your traffic, 
spots suspicious behaviour, and explains it in plain English.

![Python](https://img.shields.io/badge/Python-3.11-blue)
![License](https://img.shields.io/badge/License-MIT-green)
![Status](https://img.shields.io/badge/Status-Active_Development-brightgreen)

---

## What it does

SecureIntel monitors DNS and network traffic, uses an ML model (Isolation Forest) 
to flag anomalies, and then calls an LLM to explain each alert in plain English — 
no technical jargon.

---

## Who is this for?

Small businesses · Home network owners · Schools & NGOs · Anyone without a dedicated IT team
